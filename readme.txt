■テーマとのマッチングの為に必要な表現すべき要素
　加工した建材を組み合わせることで、複数のパーツを組み合わせた折り紙のように建物を作り上げる「折り紙工法」
　折り紙の持つ芸術性と実用性の性質の表現
　割れ窓理論にある、人々の行動をも変える「デザインが持つ力」
　

■操作方法
　LeapMotionを用いたジェスチャー

■世界観とストーリー　
　無骨な建物　折り紙　魔法　アート　デザイン　変形　合体　
　指揮者として使い魔を操る
　

■ジャンル
　マジカルリズムアクション

■システム
　初心者のモチベーションが持続できる
　中毒性がある
　リプレイしたくなる
　１プレイの時間が短い
　１プレイで満足できない
　　次にすべきことがわかる（反省して学べる）
　バランスブレイカー
　折り紙の楽しさ
　LeapMotionで実現できる非現実性
